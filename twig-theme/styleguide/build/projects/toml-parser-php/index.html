Toml parse for PHP

A PHP parser for TOML compatible with TOML v0.4.0.

Installation

Use Composer to install Yosyfmony Toml package:

Add the following to your composer.json and run composer update.

"require": {
    "yosymfony/toml": "0.3.*"
}
